Cool Currency Converter

Cool Currency Converter

Cool currency converter is a jQuery plugin run with php functions to create a simple and a cool currency conversion widget.

You can try it

Install

There are a few requirements you need to meet before you can use cool currency converter.
You need to include font-awesome, less-compiler, and our less file into your html head 

<link rel="stylesheet" href="ccc-webroot/css/font-awesome-4.3.0/css/font-awesome.min.css">
<link rel="stylesheet/less" href="ccc-webroot/css/ccc-styles.less" type="text/css">
<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/2.5.1/less.min.js"></script>

Then you need to call jQuery, and Cool Currency Converter JS file in the end of your html body

<script type="text/javascript" src="ccc-webroot/js/jquery-1.9.1.min.js"></script>
<script type="text/javascript" src="ccc-webroot/js/ccc-1.0.js""></script>

PHP

You need to add this php code to your php file to be able to call cool currency converter

<?php
    require_once "CoolCurrencyConverter.php";
    $ccc = new CoolCurrencyConverter();
?>

You need to add this php code in your page to add the cool currency converter

<?php
    $ccc->create(
    	array(
    	    "template"=>"vertical", //["horizontal", "vertical"]
            "color"=>"green", //["blue", "green", "orange", "red", "custom"]
            "border"=> true //you can remove border by making "border" => false
    	)
    ); 
?>

Template:
by default template is vertical and you can choose between Horizontal and vertical oriontaions

color:
by default color is blue and you can chose between(blue, green, orange, red). You can chouse you custom color by choosing "custom" as color, but then you have to go edit the ccc-styles.less file located in ccc-webroot/css/ and modify the values of custom colors:

@color-custom		: #8b2bdc;
@color-custom-rgb-3	: rgba(139, 43, 220, .3);
@color-custom-rgb-7	: rgba(139, 43, 220, .7);

border:
by default border is true and you can remove it by turning border to false

Ajax call

Cool currency code use ajax to get the currency result trow a php file"ajax.php". If you need to change the ajax call url u can edit the "ccc-1.0.js" file located on "ccc-webroot/js/" and edit the ajax_url config
